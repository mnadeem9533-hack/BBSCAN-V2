# BBSCAN-V2
BBScan is a Python-based, modular web application security scanner designed for educational  and lab environments. Features include intelligent URL crawling, auto parameter discovery,  SQL Injection &amp; XSS scanning, WAF detection, payload management, multi-threaded scanning,  and HTML/JSON report generation. 
sudo git clone https://github.com/mnadeem9533-hack/BBSCAN-V2.git 
cd bbscan
pip install -r requirements.txt
 usage like bbscan --url https://example.com --threads 10 --deep 2 --report myreport.html
